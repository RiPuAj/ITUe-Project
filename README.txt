# GlovoClone Project Setup Instructions

Welcome to the GlovoClone project! To get started, please follow these simple setup instructions:

## Initial Setup
First and foremost, we need to prepare our environment:
1. In the `GlovoClone` folder, execute the command: 
   ```
   npm install
   ```
2. Then, navigate to the frontend directory:
   ```
   cd ./frontend/
   ```
3. Inside the frontend directory, again run:
   ```
   npm install --force
   ```

## Running the Application
To use the app, you will need to have two terminals open:
1. In the first terminal, navigate to the `GlovoClone` directory and run:
   ```
   npm run dev
   ```
2. In the second terminal, go to the `GlovoClone/frontend` directory and execute:
   ```
   npm run dev
   ```

That's it! You are now ready to explore and use the GlovoClone app. Happy coding!
